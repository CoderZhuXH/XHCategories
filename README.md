# XHCategories
* 简洁实用的类扩展

## 使用方法
*  详见对应头文件所提供方法

##  安装
### 手动添加:<br>
*   1.将 XHCategories 文件夹添加到工程目录中<br>
*   2.导入 XHCategories.h

### CocoaPods:<br>
*   1.在 Podfile 中添加 pod 'XHCategories'<br>
*   2.执行 pod install 或 pod update<br>
*   3.导入 XHCategories.h

##  系统要求
*   该项目最低支持 iOS 7.0 和 Xcode 7.0
